## 🚀 如何使用

### 启动项目

1. **启动后端**
```bash
cd backend
npm install
npm run dev
```
后端运行在：http://localhost:3000

2. **启动前端**
```bash
cd frontend
npm install
npm run dev
```
前端运行在：http://localhost:5174

### 登录管理后台

1. 访问：http://localhost:5174/admin/login
2. 用户名：admin
3. 密码：admin123
