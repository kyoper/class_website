## 管理员使用指南

### 登录管理后台

1. 访问：`/admin/login`
2. 输入用户名和密码
3. 点击登录

**默认账号：**
- 用户名：admin
- 密码：admin123

⚠️ **重要：首次登录后请立即修改密码！**

---

### 仪表板

登录后会看到仪表板，显示：
- 各类内容的统计数据
- 快捷操作按钮
- 内容管理菜单

---

### 公告管理

#### 发布公告

1. 点击侧边栏"公告管理"
2. 点击"发布公告"按钮
3. 填写表单：
   - **标题**：公告标题（必填）
   - **摘要**：简短描述（可选，不填会自动生成）
   - **内容**：使用富文本编辑器编写内容
   - **重要**：是否标记为重要公告
4. 点击"发布公告"

#### 编辑公告

1. 在公告列表中找到要编辑的公告
2. 点击"编辑"按钮
3. 修改内容
4. 点击"更新公告"

#### 删除公告

1. 在公告列表中找到要删除的公告
2. 点击"删除"按钮
3. 确认删除

---

### 相册管理

#### 创建相册

1. 点击侧边栏"相册管理"
2. 点击"创建相册"按钮
3. 填写：
   - **相册名称**：例如"春游照片"
   - **相册描述**：简短介绍
   - **封面图片URL**：可选
4. 点击"创建"

#### 添加照片

1. 在相册列表中点击"管理照片"
2. 点击"添加照片"按钮
3. 填写：
   - **照片URL**：图片的网址
   - **照片描述**：可选
4. 点击"添加"

**获取图片URL的方法：**
- 使用图床服务（imgur、sm.ms）
- 上传图片后复制链接
- 粘贴到表单中

#### 设置封面

1. 在照片管理页面
2. 找到想要设为封面的照片
3. 点击"设为封面"

#### 删除照片

1. 在照片管理页面
2. 点击照片下方的"删除"按钮
3. 确认删除

---

### 成员管理

#### 添加成员

1. 点击侧边栏"成员管理"
2. 点击"添加成员"按钮
3. 填写：
   - **姓名**：成员姓名
   - **角色**：教师或学生
   - **职位/学号**：例如"班主任"或"001"
   - **简介**：个人介绍
   - **头像URL**：头像图片链接
   - **排序**：数字越小越靠前
4. 点击"添加"

#### 编辑成员

1. 在成员列表中找到要编辑的成员
2. 点击"编辑"按钮
3. 修改信息
4. 点击"更新"

---

### 课程表管理

#### 编辑课程

1. 点击侧边栏"课程表管理"
2. 点击要编辑的课程格子
3. 填写：
   - **科目**：从下拉列表选择或手动输入
   - **教师**：任课教师（可选）
   - **教室**：上课教室（可选）
4. 点击"更新"

#### 保存课程表

编辑完成后，点击右上角"保存课程表"按钮。

---

### 作业管理

#### 发布作业

1. 点击侧边栏"作业管理"
2. 点击"发布作业"按钮
3. 填写：
   - **日期**：作业日期
   - **科目**：例如"语文"
   - **作业内容**：详细描述
   - **截止日期**：可选
   - **附件链接**：可选
4. 点击"发布"

#### 编辑/删除作业

- 点击"编辑"修改作业
- 点击"删除"删除作业

---

### 荣誉管理

#### 添加荣誉

1. 点击侧边栏"荣誉管理"
2. 点击"添加荣誉"按钮
3. 填写：
   - **荣誉名称**：例如"优秀班集体"
   - **类别**：学术/体育/艺术/其他
   - **获得者**：个人或集体
   - **日期**：获得日期
   - **描述**：详细说明
   - **证书图片URL**：可选
4. 点击"添加"

---

### 留言管理

#### 查看留言

1. 点击侧边栏"留言管理"
2. 查看所有留言列表

#### 删除留言

**单个删除：**
- 点击留言后的"删除"按钮

**批量删除：**
1. 勾选要删除的留言
2. 点击"批量删除"按钮
3. 确认删除

---

### 班级信息管理

#### 编辑班级信息

1. 点击侧边栏"班级信息"
2. 填写/修改：
   - **班级名称**：例如"七（三）班"
   - **班级口号**：例如"团结奋进"
   - **班级简介**：详细介绍
   - **班主任**：班主任姓名
   - **学生人数**：班级人数
   - **成立时间**：例如"2023"
   - **背景图**：首页横幅背景
   - **Logo**：班级标志
   - **联系方式**：邮箱和电话
3. 点击"保存信息"

## 常见问题

### Q1: 忘记管理员密码怎么办？

**方法一：重置数据库**
```bash
cd backend
npm run seed
```
这会重置为默认密码 admin123

**方法二：直接修改数据库**
使用数据库工具修改 Admin 表中的密码字段。

---

### Q2: 如何上传图片？

目前系统使用图片URL方式：

1. **使用图床服务**（推荐）
   - 访问 imgur.com 或 sm.ms
   - 上传图片
   - 复制图片链接
   - 粘贴到系统中

2. **使用 GitHub**
   - 将图片放在 `public/images` 目录
   - 使用相对路径：`/images/photo.jpg`

---

### Q3: 富文本编辑器如何使用？

**基本功能：**
- 标题：选择标题级别
- 加粗：选中文字点击 B
- 斜体：选中文字点击 I
- 列表：有序/无序列表
- 颜色：文字颜色和背景色
- 对齐：左对齐/居中/右对齐
- 链接：插入超链接
- 图片：插入图片URL

---

### Q4: 如何备份数据？

**手动备份：**
1. 复制 `backend/prisma/dev.db` 文件
2. 保存到安全位置

**定期备份：**
建议每周备份一次数据库文件。

---

### Q5: 网站访问慢怎么办？

**优化建议：**
1. 使用图床CDN加速图片
2. 压缩图片大小（建议<2MB）
3. 清理不必要的旧数据
4. 考虑升级服务器

---

### Q6: 如何修改网站样式？

**修改颜色：**
编辑 `frontend/src/index.css` 中的 CSS 变量

**修改布局：**
编辑对应的组件文件

---

### Q7: 可以添加更多管理员吗？

目前系统只支持一个管理员账号。如需多管理员功能，需要修改代码。

---

### Q8: 留言板有审核功能吗？

目前留言直接发布，管理员可以在后台删除不当留言。

---

### Q9: 如何导出数据？

**导出数据库：**
```bash
cd backend
npx prisma studio
```
在 Prisma Studio 中可以查看和导出数据。

---

### Q10: 系统支持多少数据量？

**建议上限：**
- 公告：1000 条
- 相册：100 个
- 照片：5000 张
- 成员：200 人
- 留言：10000 条

超过建议值可能影响性能。

## 💡 使用技巧

### 1. 内容规划

- 定期发布公告（每周至少1条）
- 及时更新作业信息
- 整理照片到不同相册
- 记录班级荣誉

### 2. 图片管理

- 统一使用图床服务
- 图片命名规范
- 定期清理无用图片
- 保持图片质量和大小平衡

### 3. 数据安全

- 定期备份数据库
- 使用强密码
- 不要分享管理员账号
- 定期检查留言内容

### 4. 用户体验

- 公告标题简洁明了
- 照片添加描述
- 作业内容清晰
- 及时回应留言
